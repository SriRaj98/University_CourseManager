Instructions to run the project : 

•	Install Xampp
•	Download www40 folder from elearning and unzip.
•	Import www40.sql file from the folder into PhpMyAdmin
•	Start Xampp and run index.php in www40 folder to start the course manager
•	Login with admin1 and admin1 as userid and password.
•	You can add, update, delete and sort .
